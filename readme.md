# Dự án Đi Đâu Đây

## Hướng dẫn thực hiện git :

### 1. Clone repo :
``
    git clone https://github.com/InvisK/didauday.git
    git checkout develop
``

### 2. Update composer :
``
    composer update
``
### 3. Create .env file from .env.example and update values accordingly
``
    cp .env.example .env
``
